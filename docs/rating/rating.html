Rating

Introduction

Ratings are a good way for the user to share their feedback or opinion on anything.


ClutchUI provides a easy way to implement ratings.

Simple Rating

You can use the rating-filled class to fill the current star.


Additionally you can use the rating-half class to half-star effect.


Make sure to use the rating-container class to the parent container.
