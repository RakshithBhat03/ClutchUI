Badge

Introduction

Badges are commonly used to display status information such as notification count or user status.

Badge for Avatars

Badges on avatars can be easily implemented with ClutchUI avatars integration. Just add the class badge-avatar to parent container of the avatar.


Note: The <span> element inside the container will have the class badge which can be used to set number values.

Badge Sizes

ClutchUI provide 4 types of badge sizes.


<badge--sm | badge--md | badge--lg | badge--xl>

Badge positioning

Badges can be postioned with postioning utility classes.


<badge--top-left | badge--top-right | badge--bottom-left | badge--bottom-right>

Badge with icon

Badge on icon are commonly used as notification counts and also for cart count in e-commerce sites.

Badge with status

Badges with status are commonly used to display the stauts of the user. Most commomly used on avatars in social media sites. ClutchUI provides 4 types of status badges.
