Avatar

Introduction

An avatar is mini user profile image which is used to give an identity to the user.

Round avatar

A simple round avatar is a users profile image in the shape of circle. This can be implemented by using the avatar class to the parent contaienr of your image. In ClutchUI, by default all avatars will be round.


Note: The image should also contain the class avatar__img within it. The image should have the parent container of class avatar.

Avatar Size

ClutchUI provides 5 different sizes for you avatar.


<avatar-xs | avatar-sm | avatar-md | avatar-lg | avatar-xl>

Square avatar

To make your avatar square in shape, add avatar__img--square to your image classlist.

Rounded avatar

To make your avatar rounded on the corners, just add avatar__img--rounded to your image classlist.
