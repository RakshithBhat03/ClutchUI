Image

Introduction

Images are everywhere on the internet. With ClutchUI, implementing images so much more easier than before.

Round Image

You can make your image round by using the img--round class in your image tag.

Square Image

You can make your image square by using the img--square class in your image tag.

Cornered Image

You can make your image have round borders by using the img--corner class in your image tag.

Responsive Image

Responsive images is made easy with ClutchUI. You can use the img--responsive class in your image tag.
