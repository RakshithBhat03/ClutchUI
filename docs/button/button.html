Button

Introduction

Buttons are essential part of any website. They are commonly used to interact and get feedback from the user. With ClutchUI you can build various types of buttons.

Button sizes

ClutchUI provides 4 different sizes of buttons.


<btn--sm | btn--md | btn--lg | btn--xl>

Types of buttons

ClutchUI provides 5 types of buttons which can be used for various purposes.


<btn--primary | btn--secondary | btn--success | btn--warning | btn--danger>

Buttons with outline

ClutchUI provied buttons with borders. It can be implemented by using btn--outline class.

Disabled buttons

Create disabled buttons with the help of btn--disabled class. Buttons have cursor: not allowed property set that disables any cursor events.

Buttons with link and icon

ClutchUI provides buttons as links and also buttons with icon. It can be implemented using btn--link and btn--icon.
